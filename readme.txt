Bookmark Me is a plugin to add bookmark buttons (even non english bookmark sites) to your post or page.

== Description ==

A simple and light plugin to add social bookmarks buttons to
post and pages. I'm focusing in collecting non-english bookmarks services. 


== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
